# klatin

Enumerate k-latin squares of order n.

Debian Squeeze:

    sudo apt-get install mpich-bin libmpich1.0-dev libboost-all-dev

Debian testing:

    sudo apt-get install mpich2 libmpich2-dev libboost-all-dev

Then:

    cd nauty25r2
    ./configure
    make
    cd ..
    ./compile
